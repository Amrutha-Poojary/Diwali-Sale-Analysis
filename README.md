# 🪔 Diwali Sales Analysis

## 📌 Project Overview
This project analyzes Diwali sales data to understand customer purchasing behavior and identify key sales trends. The analysis helps businesses make data-driven decisions for marketing, customer targeting, and sales strategies.

## 🎯 Objectives
- Clean and preprocess sales data.
- Analyze customer demographics.
- Identify top-performing products and categories.
- Discover sales trends and customer preferences.
- Generate business insights through data visualization.

## 📊 Dataset
The dataset contains information about:
- Customer ID
- Gender
- Age Group
- State
- Occupation
- Product Category
- Orders
- Amount Spent

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔍 Analysis Performed
- Data Cleaning and Preprocessing
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Customer Demographic Analysis
- Product Category Analysis
- Sales Trend Analysis
- Data Visualization

## 📈 Key Insights
- Married women aged 26–35 years were the major buyers.
- Customers from IT, Healthcare, and Aviation sectors contributed significantly to sales.
- Certain product categories generated higher revenue during the Diwali season.
- Sales were concentrated in specific states and regions.

## 📷 Visualizations
The project includes:
- Bar Charts
- Count Plots
- Sales Distribution Charts
- Customer Demographic Analysis Graphs

## 🚀 How to Run
1. Clone the repository.
2. Install the required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook.
4. Run all cells to view the analysis and visualizations.

## 📌 Conclusion
This project provides valuable insights into customer buying behavior during the Diwali season. The findings can help businesses improve customer targeting, optimize marketing campaigns, and increase sales performance.
